# FluxTune

An Arduino-based ham radio simulator that provides a realistic experience of tuning across radio bands and hearing CW (Morse) and RTTY stations.

## Features

- Realistic VFO tuning with rotary encoder
- Digital frequency display
- Simulated CW (Morse code) stations
- Simulated RTTY stations
- Asynchronous multi-station simulation
- LED signal indicators
